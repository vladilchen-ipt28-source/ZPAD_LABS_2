Опис виконання
1. Генерація сигналу: Створено функцію для моделювання гармонічного коливання $y(t) = A \cdot \sin(\omega t + \varphi)$ з можливістю накладання випадкового гаусового шуму.
2. Інтерактивний інтерфейс: Розроблено систему керування параметрами в реальному часі за допомогою слайдерів, кнопок та чекбоксів бібліотеки Matplotlib.
3. Фільтрація: Реалізовано механізм цифрової обробки сигналу для очищення зашумленої гармоніки та відновлення її первісного вигляду за допомогою фільтра Савицького-Голея.
4. Порівняльний аналіз: Програма відображає два графіки одночасно — оригінальний зашумлений сигнал та результат роботи фільтра для візуальної оцінки якості обробки.

Детальний опис програмного коду
Програма написана мовою Python у середовищі Jupyter Notebook із використанням бібліотек NumPy, Matplotlib та SciPy.

Блок математичних розрахунків
Функція `harmonic_with_noise`:** Обчислює значення синусоїди та додає шум за допомогою `np.random.normal`. Реалізовано логіку збереження поточного шуму: при зміні параметрів гармоніки масив шуму залишається незмінним.
Фільтрація: Використано фільтр `savgol_filter` з пакету `scipy.signal`. Він ефективно згладжує високочастотний шум, зберігаючи при цьому основну форму сигналу.

Блок інтерактивного керування
1. Слайдери (Sliders): Дозволяють змінювати амплітуду, частоту, фазу гармоніки, а також математичне очікування та дисперсію шуму.
2. Чекбокс (CheckButtons): Керує видимістю зашумленої кривої на графіку.
3. Кнопка Reset: Автоматично повертає всі інтерактивні елементи та графіки до початкових значень.
4. Функція `update`: Виконує перерахунок даних та перемальовування графіків (`canvas.draw_idle`) при кожній зміні положення слайдерів.
