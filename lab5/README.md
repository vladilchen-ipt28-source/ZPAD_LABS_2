Загальний опис проекту
У цій роботі я розробив повноцінний веб-додаток для інтерактивного аналізу показників вегетації. За основу я взяв дані, які збирав та обробляв у Лабораторній роботі №2 (індекси VCI, TCI та VHI по областях України). Мій додаток дозволяє користувачу зручно фільтрувати ці дані та переглядати графіки прямо у браузері без необхідності запускати складний код чи відкривати таблиці вручну.

Технічні характеристики та архітектура
Для розробки я використав мову Python та фреймворк Streamlit. Візуалізацію я реалізував за допомогою бібліотеки Matplotlib, а всю роботу з таблицями — через Pandas. Мій код побудований за модульним принципом:

1. Завантаження та обробка даних
Я реалізував функцію, яка автоматично сканує папку vhi_data та об'єднує всі знайдені CSV-файли в одну таблицю. Щоб додаток працював швидше, я додав кешування через декоратор cache_data — це дозволяє не перечитувати файли щоразу, коли користувач змінює фільтри.

2. Інтерфейс керування
У лівій частині додатка я розмістив панель налаштувань, де користувач може:
- Вибрати потрібний індекс (VCI, TCI або VHI).
- Обрати конкретну область для аналізу.
- Виставити діапазон тижнів та років за допомогою слайдерів.
- Увімкнути сортування даних за зростанням або спаданням.
- Скинути всі налаштування до початкового стану однією кнопкою.

3. Візуалізація та аналітика
Основний простір додатка я розділив на три вкладки:
- Таблиця: тут я вивів відфільтровані дані у чистому вигляді.
- Графік: тут відображається динаміка обраного індексу саме для тієї області, яку вибрав користувач.
- Порівняння: на цій вкладці я реалізував графік, де можна порівняти показники обраної області з усіма іншими областями України одночасно.

Інструкція з розгортання та використання
Щоб запустити мій проект, потрібно виконати наступне:

Встановлення бібліотек:
Відкрийте термінал та встановіть необхідні пакети:
pip install streamlit pandas matplotlib

Запуск програми:
Перейдіть у папку з моїм проектом і запустіть сервер командою:
streamlit run lab5.py

Після цього додаток автоматично відкриється у вашому браузері.
